# Known Announcement → Issue Mappings

This file tracks community.ta-39.com/announcements posts and the GitHub issues
they correspond to. It is the input for the **Announcement Cross-Reference**
table in `TA39-Roadmap.md` and for assigning the `[ANNOUNCED]` / `[FEATURED]`
tags.

**Rules:**
- `[ANNOUNCED]` applies if there is a dedicated post below.
- `[FEATURED]` applies if the item is released (`Testing in Production` OR
  `In Production & Done`) AND is named on www.ta-39.com/en/features.
  Pipeline items never carry `[FEATURED]` even if they appear on the features
  page — that is a commitment, not a claim.
